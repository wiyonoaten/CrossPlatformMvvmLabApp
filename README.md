XPlatMvvmLabApp
===============

The ultimate showroom for demonstrating cross-platform native app development using MVVM technique and Xamarin technologies.

Frameworks used are:
* MVVM Light
* MvvmCross (for data binding only)
* Xamarin.Forms

This project showcases following different types of apps:
1a) Shared C# business logic + Shared C# view model + Android-specific C# ui code (Android native app)
1b) Shared C# business logic + Shared C# view model + iOS-specific C# ui code (iOS native app)
1c) Shared C# business logic + Shared C# view model + WinPhone-specific C# ui code (WinPhone native app)
2)  Shared C# business logic + Shared C# view model + Shared Xamarin.Forms-based C# ui code (Android native app + iOS native app)
